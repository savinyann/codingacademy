Make them aware of your power

This file must contain a function which assigns a "click" event on the first "button" element of the page. The action of the event must make all paragraph of the page disappear.

I have made two button.

One of them always tell the truth,

When the other one always lie.

One of them destroy all "p",

When the other one let them live.

Find which one let the "p" live. Or destroy them, as you want.
